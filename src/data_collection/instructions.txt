Instructions
1. Populate the .env file with your credentials
  - create a .env file containing this 
EMAIL=*insert*
EMAIL_PASSWORD=*insert*
USERNAME=*insert*
PASSWORD=*insert*
2. Change the directory of dotenv_path in headless_v2.py to point at your .env file
3. Change the observer.bat python / to point to your directory
4. Run observer.bat as administrator
